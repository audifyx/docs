# OGSCAN FULL EXPLANATION

# WHAT IS OGSCAN?

OGSCAN is a launch-intelligence system focused on reconstructing launch history across Solana ecosystems.

Traditional scanners normally index:
- first visible pair
- first indexed contract
- first liquidity event

OGSCAN instead attempts to reconstruct:
- token lineage
- launch continuity
- migration behavior
- deployer relationships
- liquidity sequencing
- wallet ecosystem behavior

This creates a fundamentally different analysis system.

---

# WHY OGSCAN MATTERS

Solana launches are extremely fragmented.

Common problems include:
- duplicated ticker launches
- fake OG tokens
- migration confusion
- relaunch cycles
- cloned contracts
- bundled launches
- fake community launches
- liquidity migration
- spam deployment flooding

Traditional scanners often incorrectly identify the wrong token as the canonical launch origin.

OGSCAN attempts to determine the highest-probability original ecosystem using multiple ecosystem signals simultaneously.
